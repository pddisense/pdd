Private Data Donor

Chrome extension is monitoring Google search engine use patterns. Search engine queries are saved and periodically submitted to our server. Currently the extension works in NoPrivacy mode, which means that privacy-preserving protocols are not used.

Our server receives raw search engine queries and links the user has clicked on. The submission to server takes place over an encrypted channel.

Warning. After a restart of Google Chrome, the browser disables extension not present in Chrome App Store. To mitigate this issue, make sure you are using Chromium.

How to install the extension

Caveat. In development mode, we did not submit the extension to Chrome App Store.

  1. Download Chromium browser from here and then just navigate to sdaq.crx. This version of Chromium is not disabling extensions. The extension will be kept after a browser restart.
  2. Click on the link to extension here.
  3. It should now work. Open e.g. www.google.com. Search for something. Submissions to the server are periodic.
  4. Right-click on the i-sense button in the browser toolbar. Look at 'Options' or 'Edit demography data'.
